# Описания файлов

filter.ipynb - фильтровка данных (убираем лишний критерии)
Final_Conclusion.ipynb - финальный вывод данных
ML.ipynb - обучение модели и её характеристики
unpacking.ipynb - скачивание и распаковка данных
users.ipynb - определение холодного пользователя и его характеристики

user0_city.parquet - принадлежность пользователя к городу по процентам
user0_tags.parquet - принадлежность пользователя к тегам по процентам
user0_regions.parquet - принадлежность пользователя к региону по процентам

random_forest_model.joblib - обученная модель

security.parquet - показанные пользователю видео

logsV02.parquet - выжимка значений для выдачи
final_videos_info.parquet - информация о всех видео

final_videos_info.json - вывод информации о 10 видео

в папке ex - базовое описание пользователя
в папке parquet-filtered - отфильтрованные значения логов
